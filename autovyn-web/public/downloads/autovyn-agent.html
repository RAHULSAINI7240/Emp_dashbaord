Linux

Package (.deb) or portable app (.AppImage)

Ubuntu and Debian users should prefer the .deb installer. If you use the .AppImage, open a terminal and run these commands first:
sudo apt install libfuse2 (one-time setup)
chmod +x Autovyn-Desktop.AppImage
./Autovyn-Desktop.AppImage
Without these steps Linux will block the file from running.

After download: On Windows, double-click the .exe file to install. On macOS, open the .dmg and drag the app to Applications.
