Install steps

Install in three steps

1 Download the installer that matches the employee computer.
2 Open the downloaded file and finish setup like a normal desktop app.
3 Open the app, sign in once, and leave auto-start enabled if you want background tracking.
Mac note

If macOS says This application is not supported on this Mac, the wrong Mac build was downloaded. Use Apple Silicon for M1, M2, M3, or M4 Macs and Intel for older Macs.
